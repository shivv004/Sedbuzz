# Sedbuzz

Sedbuzz is a blog project inspired by the blog "Wired".

## Installation

Use the package manager [npm](https://pip.pypa.io/en/stable/) to install the required dependencies.

```bash
npm i

#client
npm start

#server(nodemon)
npm run dev

#server(node)
npm start
```
